# Netflix Clone – HTML & CSS Project

This project is a front-end clone of the popular streaming platform **Netflix** built using **HTML5 and CSS3**.  
The goal of this project was to practice modern UI design, layout structuring, and responsive web development.

The project recreates the **Netflix landing page interface**, including the hero section, trending movies slider, FAQ section, and footer.

---

## Features

- Netflix style navigation bar
- Hero section with background image
- Email signup input and button
- Trending movies horizontal slider
- Hover animation on movie cards
- "More reasons to join" feature section
- Frequently Asked Questions accordion
- Responsive footer with multiple links
- Fully responsive layout (mobile, tablet, desktop)

---

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Google Fonts
